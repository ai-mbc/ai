CHAPTER 02

자료형과 연산자

조수를 채용했으니 이제 일을 시킬 차례예요. 김밥이 3500원, 손님이 2줄 주문 — 이런 값을 어딘가에 담아 두고 합계를 계산하게 해야죠. 값을 담는 '변수', 값의 종류인 '자료형', 계산 규칙인 '연산자'를 배웁니다.

🍜 파이 분식 이야기 — 2화. 계산기부터 만들자

첫 손님이 왔어요. "김밥 2줄, 라면 1개요!" 사장님은 조수에게 계산을 맡기고 싶습니다. 그러려면 먼저 가격표가 있어야 하고(=변수에 값 담기), 가격이 숫자인지 글자인지 구분해야 하며(=자료형), 곱하고 더하는 규칙이 필요해요(=연산자). 이번 장을 끝내면 조수가 주문 금액을 척척 계산하게 됩니다.

🎯 이 장을 끝내면 이런 걸 할 수 있어요

  • 변수를 만들고 값을 저장할 수 있다
  • 정수·실수·문자열·논리형을 구분하고 변환할 수 있다
  • input()으로 입력받고 print()로 출력할 수 있다
  • 문자열을 자르고(슬라이싱) 이어 붙일 수 있다
  • format()으로 값을 원하는 형식으로 꾸밀 수 있다
  • 산술·비교·논리 연산자로 계산할 수 있다

1변수 — 값을 담는 상자

📦 변수(Variable)
값을 저장해 두고 이름을 붙여 두는 상자예요. 프로그램이 도는 동안 값이 바뀔 수 있어서 '변수(변하는 수)'라고 부릅니다.
🏷️ 비유 — 주방 선반에 '김밥가격'이라고 이름표를 붙인 통을 두는 것과 같아요. 나중에 "김밥가격 통 가져와"라고 이름만 부르면 그 안의 값(3500)이 나옵니다.

왜 변수가 필요할까요? 가격을 매번 3500이라고 직접 쓰면, 가격이 오를 때 코드 곳곳을 다 고쳐야 해요. 하지만 김밥가격이라는 상자에 담아 두면 한 군데만 바꿔도 전부 반영됩니다. 파이썬에서 변수 만들기는 이름 = 값 한 줄이면 끝이에요.

python
김밥가격 = 3500        # '김밥가격' 상자에 3500을 넣음
주문수량 = 2           # 손님이 김밥 2줄 주문
합계 = 김밥가격 * 주문수량   # 곱해서 합계 계산
print(합계)
7000
💡 '=' 는 같다는 뜻이 아니에요
수학에서 =는 '같다'지만, 프로그래밍에서 =는 "오른쪽 값을 왼쪽 상자에 넣어라"라는 뜻(할당)이에요. 이걸 할당(대입) 연산자라고 부릅니다. 파이썬은 상자를 만들 때 자료형을 미리 정하지 않아요(선언 과정이 없음).

변수 이름 짓는 규칙

규칙바른 예 ✅틀린 예 ❌
숫자로 시작 못 함year20002000year
대소문자 구분함Class, class는 서로 다름—
특수문자·공백 불가 (밑줄 _만 허용)member_namemember.name, member name
예약어 사용 금지—for, class, def
⚠️ 예약어(keyword)란?
파이썬이 특별한 용도로 미리 찜해 둔 단어예요. if, for, def, class 같은 것들이라 변수 이름으로 쓸 수 없어요. 또 print, list 같은 내장 함수 이름도 변수로 쓰지 않는 게 좋습니다(기능이 망가져요).

변수 관련 유용한 기능

  • id(변수) — 변수가 저장된 메모리 주소(고유 번호)를 알려줘요.
  • del 변수 — 변수를 삭제해요. 여러 개는 콤마로: del a, b, c
  • 다중 할당 — 한 줄에 여러 변수를 한꺼번에: a, b = 1, 2
  • help(함수) — 함수 사용법(도움말)을 보여줘요.

2화면 입출력

⌨️ input() — 입력받기
사용자가 키보드로 값을 입력하게 해요. 중요한 점: 입력한 값은 항상 문자열(글자)로 들어옵니다.
python
name = input("이름을 입력하세요: ")
print(name + "님 환영합니다!")
이름을 입력하세요: 홍길동 홍길동님 환영합니다!
⚠️ 숫자를 입력받을 땐 변환이 필요해요
input()은 숫자를 입력해도 글자 "10"으로 받아요. 계산하려면 int()로 숫자로 바꿔야 합니다: age = int(input("나이: "))
🖨️ print() — 출력하기
괄호 안의 값을 화면에 출력해요. 여러 값을 콤마로 나열하면 띄어쓰기와 함께 이어서 보여줍니다.

print()에는 두 가지 유용한 옵션이 있어요.

  • sep — 값들 사이를 무엇으로 구분할지 (기본값: 공백 한 칸)
  • end — 출력 끝에 무엇을 붙일지 (기본값: 줄바꿈 \n)
python
print("2024", "01", "15", sep="-")   # sep로 구분
print("끝에 줄바꿈 대신", end=" → ")
print("이어서 출력")
2024-01-15 끝에 줄바꿈 대신 → 이어서 출력

3기본 자료형

🎭 자료형(Data Type)
값의 종류를 뜻해요. 숫자인지, 글자인지, 참/거짓인지에 따라 할 수 있는 일이 달라집니다.
🧊 비유 — 물의 상태(얼음·물·수증기)와 비슷해요. 같은 물이라도 상태에 따라 다루는 방법이 다르듯, 값도 자료형에 따라 쓰는 법이 달라집니다.
자료형이름예시
정수int10, -3, 0
실수(소수점)float3.14, -0.5
복소수complex3 + 2j (허수는 j로!)
문자열str"안녕", 'Python'
논리(참/거짓)boolTrue, False
💡 파이썬의 특별한 점
다른 언어와 달리 파이썬은 정수 크기에 제한이 없어요. 아무리 큰 숫자도 int로 처리합니다. 또 정수 ÷ 정수는 실수(float)가 돼요: 8 / 2 → 4.0

3-1. 문자열 다루기 핵심

문자열은 따옴표로 감싼 글자예요. 홑따옴표 ' ', 겹따옴표 " " 둘 다 됩니다.

python
s = "Python"
# 여러 줄 문자열은 따옴표 3개
long = """첫째 줄
둘째 줄"""

# 문자열 연결(+)과 반복(*)
print("Ha" + "Ha")   # 연결
print("Na" * 3)       # 반복
HaHa NaNaNa
🔢 인덱싱(Indexing) — 한 글자 뽑기
문자열의 각 글자에는 번호(인덱스)가 있어요. 첫 글자가 0번부터 시작하고, 음수는 뒤에서부터 셉니다.
python
# 인덱스:  P(0) y(1) t(2) h(3) o(4) n(5)
#         P(-6) y(-5) t(-4) h(-3) o(-2) n(-1)
s = "Python"
print(s[0])    # 첫 글자
print(s[-1])   # 마지막 글자
P n
✂️ 슬라이싱(Slicing) — 여러 글자 잘라내기
[시작:끝]으로 일부분을 잘라내요. 시작은 포함, 끝은 불포함이 핵심! [시작:끝:간격]으로 몇 칸씩 건너뛸 수도 있어요.
🎬 비유 — 영화 편집처럼 "3초부터 8초까지 잘라줘"인데, 8초 직전까지만 잘리는 규칙이라고 생각하세요.
python
s = "Python"
print(s[0:3])    # 0,1,2번 → 3번 직전까지
print(s[2:])     # 2번부터 끝까지
print(s[::-1])   # 간격 -1 → 거꾸로!
Pyt thon nohtyP
💡 [::-1] 은 뒤집기 마법
간격을 -1로 주면 문자열이 통째로 거꾸로 뒤집혀요. 리스트에서도 똑같이 작동합니다(다음 장에서!).
🚪 이스케이프 문자(escape)
역슬래시 \ 뒤에 특정 글자를 붙여 특수 기능을 나타내요.
기호의미기호의미
\n줄 바꿈\\역슬래시 표시
\t탭(간격)\'홑따옴표 표시
\r맨 앞으로 복귀\"겹따옴표 표시

경로나 정규식처럼 \를 그대로 쓰고 싶으면 문자열 앞에 r을 붙여요(raw 문자열): r"C:\새폴더"

3-2. 논리형 (True / False)

✔️ 논리형(bool)
참(True) 또는 거짓(False) 딱 두 값만 가져요. 반드시 첫 글자가 대문자예요(true는 안 됨).

어떤 값들은 자동으로 거짓(False)으로 판단돼요. 이걸 알면 조건문에서 유용해요.

⚠️ False로 취급되는 값들
None · False · 숫자 0 (0.0, 0j) · 빈 문자열 "" · 빈 리스트 [] · 빈 튜플 () · 빈 딕셔너리 {} — 이것들 외에는 모두 True예요. (참고: 공백 한 칸 " "은 비어있지 않으므로 True!)

3-3. 자료형 확인과 변환

python
print(type(10))      # 자료형 확인
print(int("100") + 5)  # 문자열 → 정수
print(str(3.14))     # 실수 → 문자열
print(float("2.5"))  # 문자열 → 실수
print(isinstance(10, int))  # 10이 int인가?
<class 'int'> 105 3.14 2.5 True
  • type(값) — 자료형이 뭔지 알려줌
  • int(), float(), str(), bool() — 자료형 변환
  • isinstance(값, 타입) — 값이 그 타입인지 True/False로 확인

4포맷팅 — 값을 예쁘게 꾸미기

🎨 포맷팅(Formatting)
문자·숫자·날짜에 형식을 지정하는 것. 소수점 자릿수, 정렬, 자리 채우기 등을 깔끔하게 표현해요.

가장 많이 쓰는 방법은 .format()과 f-문자열이에요.

python
name, age = "홍길동", 23
# format() 방식 — {}가 자리표시자
print("{}님의 나이는 {}살입니다".format(name, age))
# f-문자열 방식 — 요즘 가장 많이 씀
print(f"{name}님의 나이는 {age}살입니다")
홍길동님의 나이는 23살입니다 홍길동님의 나이는 23살입니다

숫자·정렬·소수점 형식

형식의미예시 → 결과
:d10진 정수{:d}
:f실수{:.2f} → 소수 2자리
:b :o :x2진/8진/16진수—
:< :^ :>왼쪽/가운데/오른쪽 정렬{:^10}
:05d0으로 자리 채우기42 → 00042
python
print(f"{3.14159:.2f}")    # 소수점 2자리
print(f"{42:05d}")          # 5칸을 0으로 채움
print(f"{255:x}")           # 16진수
print(f"[{'중앙':^8}]")      # 8칸 가운데 정렬
3.14 00042 ff [ 중앙 ]

날짜 기호도 알아두면 좋아요: %Y(연) %m(월) %d(일) %H(시) %M(분) %S(초)

5연산자

➕ 연산자(Operator)
값을 가지고 계산이나 비교를 수행하는 기호예요. 사칙연산부터 참/거짓 판단까지 다양합니다.

5-1. 산술 연산자

연산자의미예시 → 결과
+ - * /더하기·빼기·곱하기·나누기7 / 2 → 3.5
//몫 (버림 나눗셈)7 // 2 → 3
%나머지7 % 2 → 1
**제곱2 ** 3 → 8
💡 %(나머지)의 쓸모
어떤 수가 짝수인지 알고 싶으면 n % 2 == 0인지 보면 돼요. 배수 판별에도 자주 씁니다.

5-2. 대입(할당) 연산자

계산과 저장을 한 번에 하는 복합 대입 연산자가 편리해요.

python
x = 10
x += 5    # x = x + 5 와 같음 → 15
x *= 2    # x = x * 2 와 같음 → 30
print(x)
30

이런 식으로 +=, -=, *=, /=, //=, %=, **= 등이 있어요.

5-3. 비교 연산자

크기를 비교해 True/False로 답해요: >, <, >=, <=, ==(같다), !=(다르다)

⚠️ = 와 == 를 헷갈리지 마세요
=는 값을 넣는 것(할당), ==는 같은지 비교하는 것이에요. 초보자가 가장 많이 하는 실수랍니다!

5-4. 논리 연산자

연산자의미결과가 True일 때
and그리고양쪽 모두 참
or또는하나라도 참
not부정(반대)참↔거짓 뒤집기
⚠️ 파이썬은 !를 안 써요
C나 자바에서 쓰는 !(부정) 대신, 파이썬은 영어 단어 not을 씁니다. '그리고/또는'도 &&, ||가 아니라 and, or예요.

5-5. 비트 연산자 (참고)

숫자를 2진수로 바꿔 비트 단위로 계산해요. &(AND) |(OR) ^(XOR) ~(NOT) >> <<(시프트). 처음엔 몰라도 괜찮아요.

6문자열 메서드 모음

문자열에는 편리한 '기능(메서드)'이 많아요. 자주 쓰는 것만 추렸습니다.

메서드기능
len(s)길이(글자 수)
s.split('구분자')구분자로 잘라 리스트로
'구분자'.join(리스트)리스트를 이어 문자열로
s.upper() / s.lower()모두 대문자 / 소문자로
s.replace(옛것, 새것)바꿔치기
s.find('글자')글자 위치(인덱스) 찾기
s.startswith() / endswith()~로 시작/끝나는지 확인

s = "apple,banana,cherry"
fruits = s.split(",")     # 콤마로 자르기

['apple', 'banana', 'cherry'] apple-banana-cherry

3"Hello World"를 이용해 World Hello로 출력하세요. (분리·합치기 이용)
정답
python
s = "Hello World"
words = s.split()          # ['Hello', 'World']
print(words[1] + " " + words[0])
World Hello
해설

split()은 공백 기준으로 잘라 리스트를 만들어요. [0]은 Hello, [1]은 World이므로 순서를 바꿔 이어 붙이면 됩니다. " ".join(words[::-1])로도 가능해요.

6섭씨온도를 입력받아 화씨로 변환하세요. 단, 소수점 이하가 있을 때만 둘째 자리까지 출력하고, 없으면 소수점을 표시하지 않습니다. (F = C×1.8 + 32)
정답
python
C = float(input("섭씨온도: "))
F = C * 1.8 + 32
print(f"{F:g}")    # :g 는 불필요한 0을 자동 정리
섭씨 25 입력 → 77 섭씨 23.6 입력 → 74.48
해설

:g 형식은 필요할 때만 소수점을 보여주는 똑똑한 형식이에요. 77.0은 77로, 74.48은 그대로 보여줍니다. 정확히 둘째 자리까지 제어하려면 조건문으로 나눠 :.2f와 :.0f를 골라 쓰는 방법도 있어요.

8다음 중 파이썬 3에서 결과가 잘못 짝지어진 것은? ① 8//2 → 4 ② 8/2 → 4 ③ 8**2 → 64

② — 8/2의 결과는 4가 아니라 4.0

파이썬 3에서 /(나눗셈)는 항상 실수(float)를 만들어요. 그래서 8/2는 4.0입니다. 정수 몫이 필요하면 //를 써야 해요. (8//2 → 4, 8**2 → 64는 맞음)
