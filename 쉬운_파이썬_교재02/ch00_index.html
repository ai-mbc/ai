쉽게 다시 만든 파이썬 교재

어려운 용어는 쉬운 비유로, 개념마다 예제와 풀이로. 처음 배우는 사람도 따라올 수 있게 재구성했어요.

📚 이 교재는요

원본 강의 자료(8개 챕터)를 더 이해하기 쉽게 다시 정리했어요. 각 용어에 쉬운 정의와 실생활 비유를 붙이고, 개념마다 실행 결과가 있는 예제와 정답·해설이 담긴 연습문제를 넣었습니다.

🔤 쉬운 용어 정의 🍳 실생활 비유 💻 실행 예제 📝 정답·해설 연습문제

🍜 하나의 이야기로 배웁니다 — 파이 분식 창업기

문법을 따로따로 외우면 "이걸 언제 써먹지?" 하는 물음이 남아요. 그래서 이 교재는 작은 분식집 '파이 분식'을 파이썬으로 하나씩 자동화해 가는 이야기로 8개 챕터를 이어 붙였습니다.

가게 문을 열기로 결심하고(1장) → 메뉴 가격을 계산기에 담고(2장) → 메뉴판 전체를 관리하고(3장) → "재고 있으면 팔고 없으면 품절"을 판단하고(4장) → 반복되는 계산을 묶고(5장) → 계산 도구를 파일로 나눠 정리하고(6장) → 메뉴 하나하나를 '객체'로 만들고(7장) → 손님이 이상한 주문을 해도 가게가 멈추지 않게(8장) 만듭니다.

👉 매 장을 지날수록 '파이 분식'이 점점 똑똑해져요. 문법 하나하나가 왜 필요한지 자연스럽게 몸에 뱁니다.

💡 이렇게 공부하세요 — 챕터를 순서대로 읽으며 예제를 직접 따라 쳐 보세요. '파이 분식' 이야기가 장마다 이어지니, 1장부터 차례로 보는 걸 권해요. 연습문제는 먼저 스스로 푼 뒤 '정답·해설 보기'를 눌러 확인하면 실력이 쑥쑥 늘어요.
