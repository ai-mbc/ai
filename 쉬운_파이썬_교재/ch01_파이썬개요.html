CHAPTER 01

파이썬 개요 및 개발환경 구성

파이썬이 어떤 언어인지, 왜 배우기 쉬운지, 그리고 코드를 작성하고 실행할 환경을 어떻게 갖추는지 알아봅니다.

🎯 이 장을 끝내면 이런 걸 할 수 있어요

  • 파이썬이 어떤 언어이고 어디에 쓰이는지 설명할 수 있다
  • 파이썬의 특징을 3가지 이상 말할 수 있다
  • 컴파일 언어와 인터프리터 언어의 차이를 안다
  • 파이썬과 아나콘다를 설치하고 Hello World를 출력할 수 있다
  • 주피터 노트북에서 코드를 작성하고 실행할 수 있다

1파이썬이란?

파이썬(Python)은 1991년에 세상에 나온 프로그래밍 언어예요. 만든 사람은 귀도 반 로섬(Guido van Rossum)이고, 이름은 뱀이 아니라 영국 코미디 그룹 '몬티 파이썬'에서 따왔습니다.

🐍 프로그래밍 언어란?
사람이 컴퓨터에게 "이렇게 해줘"라고 시키기 위해 쓰는 약속된 말이에요. 우리가 한국어로 대화하듯, 컴퓨터와 대화하는 언어입니다.
🍳 비유 — 요리 레시피와 같아요. "밀가루를 넣고 → 저은 뒤 → 구워라"처럼 순서대로 지시를 적으면, 컴퓨터(요리사)가 그대로 따라 합니다.

파이썬은 이런 성격을 가지고 있어요.

  • 기본 기능은 작게, 부가 기능은 라이브러리로 — 언어 자체는 단순하게 두고, 필요한 기능은 '라이브러리(도구 모음)'를 가져다 씁니다.
  • 교육용으로 인기 — 문법이 쉬워서 처음 배우는 사람에게 알맞습니다.
  • 무료 — 누구나 공짜로 내려받아 씁니다. 공식 사이트는 python.org
💡 알아두기
파이썬은 2000년에 버전 2, 2008년에 버전 3이 나왔어요. 지금은 파이썬 3을 씁니다. 이 교재의 모든 예제도 파이썬 3 기준입니다.

2파이썬으로 무엇을 할 수 있을까?

파이썬은 정말 다양한 분야에서 쓰여요. 대표적인 것만 봐도 이렇습니다.

✅ 잘하는 것⚠️ 잘 안 쓰는 것
데이터 분석 · 통계 · 시각화시스템 프로그래밍(운영체제 등)
인공지능 · 머신러닝 · 딥러닝모바일 앱 개발
웹 프로그래밍→ 이런 분야는 C, 자바, 스위프트 같은
다른 언어가 더 적합해요
영상 인식 · 알고리즘 공부
💡 정리
파이썬은 특히 데이터와 인공지능 분야에서 사실상 표준 언어예요. 요즘 AI가 뜨면서 파이썬 인기도 함께 올라갔습니다.

3파이썬 언어의 특징

파이썬이 왜 사랑받는지, 6가지 특징으로 정리했어요.

1. 가독성 (읽기 쉬움)
문법이 간결하고, 들여쓰기로 코드 구조를 표현해서 눈으로 읽기 편해요.
📖 비유 — 잘 정리된 목차가 있는 책 같아요. 어디가 어디에 속하는지 한눈에 보입니다.
2. 확장성
전 세계 사람들이 만든 풍부한 라이브러리가 있어서, 원하는 기능을 가져다 붙이면 됩니다.
3. 접착성 (glue 언어)
C나 C++로 만든 빠른 프로그램을 파이썬에서 불러 쓸 수 있어요. 서로 다른 언어를 풀처럼 붙여준다고 해서 '접착 언어'라 부릅니다.
4. 플랫폼 독립적
한 번 작성한 코드는 윈도우·맥·리눅스 어디서든 똑같이 실행됩니다.
5. 객체지향 언어
'객체(object)'라는 개념으로 프로그램을 조립하듯 만들 수 있어요. (자세한 건 뒷 장에서!)
6. 동적 실행 (인터프리터 언어)
코드를 작성하면서 바로바로 실행해 결과를 볼 수 있어요. 실험하듯 프로그래밍할 수 있습니다.

4컴파일 언어 vs 인터프리터 언어

프로그래밍 언어는 컴퓨터가 알아듣게 '번역'하는 방식에 따라 두 종류로 나뉘어요.

📦 컴파일 언어
코드 전체를 한꺼번에 기계어(컴퓨터의 언어)로 번역해서 '실행 파일'을 먼저 만든 뒤, 그 파일을 실행해요. (예: C, C++)
📚 비유 — 책 한 권을 통째로 번역해서 번역본을 찍어내는 것. 번역(준비)은 오래 걸리지만, 완성된 번역본은 아주 빠르게 읽을 수 있어요.
🗣️ 인터프리터 언어
'인터프리터(해석기)'가 코드를 한 줄씩 읽으며 바로 실행해요. (예: Python, JavaScript, R)
🎧 비유 — 통역사가 옆에서 한 문장씩 즉석 통역해주는 것. 바로바로 대화할 수 있지만, 통째 번역본보다는 조금 느립니다.
구분컴파일 언어인터프리터 언어
번역 방식전체를 한 번에한 줄씩 즉시
실행 속도빠름상대적으로 느림
테스트매번 전체 번역 필요바로 실행해 확인 가능
예시C, C++Python, JavaScript
💡 한 줄 요약
파이썬은 인터프리터 언어라서 코드를 짜자마자 결과를 볼 수 있어요. 그래서 배우기 좋습니다.

5개발 환경 갖추기

코드를 작성하고 실행하려면 두 가지가 필요해요.

🔧 파이썬 인터프리터
파이썬 코드를 실제로 실행시켜 주는 프로그램이에요. python.org에서 무료로 내려받습니다.

Hello World 출력하기

모든 프로그래밍의 첫걸음은 화면에 글자를 띄우는 거예요. 파이썬에선 딱 한 줄이면 됩니다.

python
print("Hello World")
Hello World

print()는 괄호 안의 내용을 화면에 출력하라는 명령이에요. 코드를 입력하고 엔터를 누르면 문법 검사와 실행이 동시에 이뤄집니다.

아나콘다(Anaconda) — 데이터 분석용 종합 세트

🅰️ 아나콘다 배포판
파이썬 + 자주 쓰는 도구 1,500개 이상을 한 번에 설치해주는 '올인원 패키지'예요. 데이터 분석·머신러닝에 필요한 것들이 다 들어 있습니다.
🧰 비유 — 공구를 하나씩 사 모으는 대신, 필요한 공구가 다 담긴 '종합 공구세트'를 통째로 사는 것과 같아요.
  • Anaconda Navigator — 설치된 도구들을 관리·실행하는 창(윈도우 앱)
  • Anaconda Prompt — 아나콘다 명령을 직접 입력하는 검은 명령창
  • Jupyter Notebook — 웹 브라우저에서 코드를 짜는 도구
  • Spyder — 파이썬 전용 통합개발환경(IDE)

주피터 노트북(Jupyter Notebook)

📓 주피터 노트북
코드·글·그래프·수식을 한 문서에 담아 '셀(cell)' 단위로 실행하는 웹 도구예요. 데이터 분석에서 가장 많이 씁니다.
📝 비유 — 실험 노트 같아요. 칸(셀)마다 코드를 적고 실행해, 결과를 바로 아래에 기록하며 진행합니다.

셀을 실행하는 단축키는 이렇게 세 가지가 있어요.

단축키동작
Shift + Enter실행하고 다음 셀로 이동 (가장 많이 씀)
Ctrl + Enter실행하고 그 자리에 머무름
Alt + Enter실행하고 아래에 새 셀 추가
💡 스파이더(Spyder) 팁
스파이더에서는 #%% 기호로 셀을 구분해요. 전체 실행은 F5, 셀 실행은 Shift+Enter, 선택 영역 실행은 F9입니다.
📝 연습문제

먼저 스스로 답을 떠올린 뒤 '정답·해설 보기'를 눌러 확인하세요.

1파이썬은 컴파일 언어일까요, 인터프리터 언어일까요? 그 이유도 한 줄로 설명해 보세요.
정답

인터프리터 언어입니다.

해설

인터프리터(해석기)가 코드를 한 줄씩 읽으며 즉시 실행하기 때문이에요. 그래서 코드를 작성하는 동시에 결과를 바로 확인할 수 있고, 이 점이 파이썬을 배우기 쉽게 만듭니다.

2화면에 안녕 파이썬이라고 출력하는 코드를 작성하세요.
정답
python
print("안녕 파이썬")
안녕 파이썬
해설

print()는 괄호 안의 값을 화면에 보여주는 명령입니다. 글자(문자열)는 반드시 따옴표 " " 또는 ' '로 감싸야 해요.

3파이썬의 특징을 3가지 이상 말해 보세요.
예시 답안

① 가독성이 좋다(들여쓰기 기반) ② 라이브러리가 풍부해 확장성이 높다 ③ 플랫폼 독립적이다(어디서든 실행) ④ 객체지향 언어다 ⑤ 인터프리터 언어라 바로 실행된다 — 이 중 3개 이상이면 정답입니다.

4다음 중 파이썬이 잘 쓰이지 않는 분야는? ① 데이터 분석 ② 인공지능 ③ 모바일 앱 개발 ④ 웹 프로그래밍
정답

③ 모바일 앱 개발

해설

모바일 앱은 보통 스위프트(iOS)나 코틀린(안드로이드)으로 만들어요. 파이썬은 데이터 분석·AI·웹 쪽에 강합니다.

5주피터 노트북에서 셀을 실행하고 다음 셀로 이동하는 단축키는 무엇일까요?
정답

Shift + Enter

해설

Ctrl+Enter는 실행 후 제자리, Alt+Enter는 실행 후 아래 새 셀 추가입니다. 셋 중 Shift+Enter를 가장 자주 씁니다.
