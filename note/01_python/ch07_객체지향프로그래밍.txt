7장 객체지향 프로그래밍

* **class**: 속성(변수)과 메소드(함수)를 갖는 객체를 만들기 위한 설계도
  모든 클래스는 `Object`를 상속받는다. (`__str__(self)`, ...)

* **class 내에서는**

  * 생성자(생성자가 없으면 디폴트 생성자 자동 생성) 및 소멸자
  * 인스턴스 변수(멤버 변수) vs 클래스 변수 vs static 변수
  * 인스턴스 메소드(일반 함수) vs 클래스 메소드 vs static 메소드(static은 이미지딥러닝에서 중요)

- class 클래스명(상속받을클래스명)
